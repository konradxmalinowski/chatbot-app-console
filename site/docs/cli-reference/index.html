CLI Reference

The CLI uses the fixed session ID 1 — normal and RAG conversations are persisted in sessions/1.json. Agent mode keeps its graph state in memory. --rag and --agent cannot be combined.

Usage

# Use the provider and model configured in .env
python src/main.py

# Select a provider and optionally override its model
python src/main.py --llm-provider openai --model gpt-4o-mini
python src/main.py --llm-provider anthropic
python src/main.py --llm-provider ollama

# Ground answers in documents from docs/
python src/main.py --rag

# Enable approval-gated tools
python src/main.py --agent

# Show all CLI options
python src/main.py --help

Flags

FlagDescription
--ragRetrieve context from local documents in docs/ and cite sources. Cannot be combined with --agent.
--agentRun the LangGraph agent with approval-gated tools instead of the plain chain. Cannot be combined with --rag.
--llm-providerOverrides LLM_PROVIDER for this process: gemini, openai, anthropic, or ollama.
--modelOverrides the model name for whichever provider is active.
--helpShows all CLI options.

Exiting

Submit an empty prompt to exit the chat loop. History for normal and RAG conversations is written to sessions/1.json on exit; agent mode's graph state does not persist between runs.
