chatbot-app Docs

An educational, multi-provider conversational AI stack built with LangChain and LangGraph — a CLI, a FastAPI service, optional local RAG, and an agent whose tool calls always require human approval.

What is chatbot-app?

chatbot-app is a single-session, in-memory CLI chatbot with a pluggable chat model backend — Gemini by default, with OpenAI, Anthropic, or local Ollama also supported. It exposes the same core LangChain Expression Language (LCEL) chain through two interfaces: an interactive Rich-powered terminal client and a FastAPI REST service.

Three optional layers sit on top of the base chat flow:

  • Retrieval-augmented generation — --rag grounds answers in local documents from docs/ via a persisted ChromaDB vector store, with citations.
  • A REST API — JWT-authenticated, rate-limited endpoints for chat, RAG, and the agent, with per-session JSON-backed history.
  • A LangGraph agent — --agent gives the model three tools (web_search, calculator, read_doc), gated behind a mandatory human-in-the-loop approval step. No tool ever executes without explicit approval, in either the CLI or the API.
